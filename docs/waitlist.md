# Waitlist & Genesis Raffle

## Testnet Waitlist

MAGMA's closed beta is invite-only. Access is earned, not first-come-first-served. Participants register at magmaprotocol.xyz/waitlist and receive a priority score that determines their place in the invite queue.

### Priority Score

Priority points are awarded based on signal provided at registration:

| Signal | Points |
|--------|--------|
| Own a Seeker Phone | +50 pts |
| Hold $SKR tokens | +30 pts |
| Discord handle provided | +15 pts |
| Twitter handle provided | +10 pts |
| Referral code used | +10 pts |

A higher priority score moves a wallet higher in the invite queue. The score is transparent and calculable — there is no hidden weighting or manual curation.

### Referral Codes

Every wallet that registers receives a unique referral code. When another wallet registers using your code, you receive +10 priority points. There is no cap on referral points.

## Genesis Origin Card Raffle

The 100 Genesis Origin Cards (MGNSS) are distributed via a weighted raffle at the close of the Galxe campaign. Every registered wallet is automatically entered. Raffle tickets are accumulated based on participation:

| Action | Raffle Tickets |
|--------|---------------|
| Coming Soon form submission | 1 ticket |
| Full waitlist form completion | 2 tickets |
| Owning a Seeker Phone | +5 tickets |
| Holding $SKR | +3 tickets |
| Discord handle provided | +1 ticket |
| Twitter handle provided | +1 ticket |
| Galxe social quests (each) | +3 tickets |
| Galxe testnet quests (each) | +5 tickets |
| Galxe Magma tier quest | +8 tickets |

Ticket weights are cumulative and never decrease. A wallet that completes multiple actions accumulates tickets from each.

The raffle uses Switchboard VRF for verifiable on-chain randomness. The seed and full participant list are published after execution for public verification. The raffle runs exactly once and is irreversible.

Winners receive their Genesis Origin Card airdropped directly to their wallet. The Genesis card grants 1.1× Echo Pool weight — see [NFT Collections](/docs/nft-collections) for details.
